## 주요 기능

1. **일일 칼로리 추적** - 섭취한 음식의 총 칼로리 계산
2. **영양소 분석** - 단백질, 탄수화물, 지방 추적
3. **목표 설정** - 일일 칼로리 목표 대비 진행 상황 표시
4. **음식 추가/삭제** - 사용자 정의 음식 항목 관리

## 아키텍처 패턴

이 앱은 간단한 아키텍처 패턴을 따릅니다:

- **Models**: 앱의 엔티티를 나타내는 데이터 클래스
- **Services**: Provider를 사용한 비즈니스 로직 및 상태 관리
- **Screens**: 전체 페이지 UI 컴포넌트
- **Widgets**: 재사용 가능한 UI 컴포넌트

## 다음 단계

Flutter SDK를 설치한 후 다음 명령어로 앱을 실행할 수 있습니다:

```bash
# 의존성 설치
flutter pub get

# 앱 실행
flutter run
```

Flutter SDK 설치는 [flutter.dev](https://flutter.dev)에서 가능합니다.
